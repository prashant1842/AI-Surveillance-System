# 🚨 AI Surveillance System

## 📌 Overview

An AI-based surveillance system that performs real-time object detection using YOLOv8 and analyzes risk levels. The system also includes WiFi and Bluetooth scanning modules for enhanced monitoring.

---

## 🚀 Features

* 🎥 Real-time object detection (YOLOv8 + OpenCV)
* 🚨 Risk classification (Safe / Medium / High)
* 📊 Interactive dashboard using Streamlit
* 📸 Automatic snapshot capture on high risk
* 📁 Logging system with CSV download
* 📡 WiFi network scanning
* 📶 Bluetooth device detection

---

## 🛠 Tech Stack

* Python
* Streamlit
* OpenCV
* YOLOv8 (Ultralytics)
* PyTorch
* Pandas

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## ⚠️ Note

Camera, WiFi, and Bluetooth features work only on local systems due to hardware limitations.
